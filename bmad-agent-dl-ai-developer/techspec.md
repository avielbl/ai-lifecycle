# Capability: Technical Specification (TECHSPEC)

## Overview
Acts as a bridge between the Researcher and the Developer to lock the experiment parameters and success criteria.

## Operating Instructions
1. **Input:** Read `docs/architecture/03_Architecture.md` and `docs/design/04_Detailed_Design.md`.
2. **Contract Writing:** Define the search space, budget, and tiered success criteria.
3. **Sign-off:** Request approval from the AI Researcher.
4. **Output:** Generate `docs/techspecs/TECHSPEC_EXP_[ID].md`.
