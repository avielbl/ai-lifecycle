# Capability: Experiment Execution

## Overview
Executes training runs (EXP-* tasks) and logs all metrics to the tracking tool.

## Operating Instructions
1. **Input:** Read `docs/techspecs/TECHSPEC_EXP_[ID].md`.
2. **Setup:** Ensure data pipelines and tracking tools are initialized (Stage 5 completion).
3. **Run:** Execute the training script.
4. **Log:** Update `docs/experiments/06_Experiment_Log.md` with the tracking URL and run status.
